Amharic Braille Document Line Images

Sample line-images with thier labels

Authors: Nega Agmas Asfaw and Birhanu Hailu Belay

This dataset contains 2100 Amharic Braille documnet Line-images, in which 2000 line-images are for training and 100 line images for testing with the corresponding Ground-Truth (GT) labels.

Line-images are stored in grayscale with .Png file format and named with nubers starting from 0001 with thier equivalent line nember in the label text , and 248 characters have been included in the labels.

For example: text in the first line of label.txt is transcription for image 0001.png(first image) in line images folder, and so on.
